    Greetings!

    This website is an information guide to the PB138 course project - Tournament Manager, made by L. Gocnik and F. Drgon.

    Intro

    The web application will allow users to create, edit, view and delete tournament trees. The application will also support exporting data in XML format with statistics on won matches and win/lose ratio.

    Project Structure

    The project is structured using the ASP.NET MVC framework, with a Data Access Layer, Business Layer and a Presentation Layer. The project also utilizes responsive web design (the bootstrap framework) and will contain a login interface.

    After succesfully signing up, the user will be able to log in and generate, edit and delete tournaments, players, teams and matches.

    The application will also allow the user to export tournament data in XML format with statistics concerning won matches.
